# Device Management

With Proxus, you can easily add your factory devices to the system, monitor their status, and manage them efficiently.

## What Does It Do?

Proxus device management helps you with:

- Connecting PLCs, sensors and other industrial devices
- Monitoring device data in real-time
- Tracking device status and performance
- Quickly detecting connection issues
- Easily configuring device settings

## Getting Started: Adding a New Device

To add a new device to the system:

1. Go to the **Devices** menu
2. Click the **New** button
3. Fill in the required fields:
- Device Name (e.g. "Line 1 PLC")
- Device Type (e.g. "PLC/SCADA")
- Location (e.g. "Production Area")
4. Click **Save**

<Tip>
    When naming devices, we recommend using names that reflect location and function.
    Example: "ProductionLine1_PLC", "PackagingArea_Sensor1"
</Tip>

## Monitoring Device Information

On the device details page, you can see:

### Basic Information
- Device status (Active/Inactive)
- Last communication time
- Data received amount
- Connection quality

### Performance Metrics
- Total request count
- Successful requests
- Failed requests
- Average response time

### Data Quality
- Accuracy rate
- Freshness rate
- Consistency level

## Common Issues

### 1. Device Connection Problems
If device shows "Not Connected":
- Ensure the device is powered on
- Check network connection
- Verify correct IP address is entered

### 2. No Data Received
If device is connected but no data coming:
- Check if data points are correctly defined
- Review scanning rate settings
- Ensure correct device profile is selected

### 3. Slow Data Transfer
If data is coming with delays:
- Check network performance
- Optimize scanning frequency
- Remove unnecessary data points

## Best Practice Tips

1. **Regular Checks**
- Check device status daily
- Review performance metrics weekly
- Address connection issues immediately

2. **Proactive Monitoring**
- Set alarms for important metrics
- Configure email notifications
- Schedule regular reports

3. **Security**
- Use strong passwords
- Review access permissions regularly
- Keep track of firmware updates

## FAQs

<AccordionGroup>
    <Accordion title="How long does it take to add a new device?">
        Adding a basic device typically takes 5-10 minutes. This includes entering device information and basic configuration. More complex devices might require additional setup time.
    </Accordion>

    <Accordion title="Can I connect multiple devices at once?">
        Yes! Proxus supports bulk device addition. You can import multiple devices using our template spreadsheet or through the API.
    </Accordion>

    <Accordion title="What happens if a device loses connection?">
        Proxus automatically detects connection losses and attempts to reconnect. You'll receive notifications based on your alert settings, and the system will log the event.
    </Accordion>

    <Accordion title="How can I improve device data quality?">
        To improve data quality:
        - Optimize scanning intervals
        - Ensure stable network connection
        - Use appropriate device profiles
        - Regular maintenance and checks
    </Accordion>
</AccordionGroup>

# Key Features

Proxus offers a comprehensive suite of features designed to handle the complete lifecycle of industrial data management. Here's a detailed look at the key capabilities that make Proxus stand out.

## Data Collection & Protocol Support

<Card title="50+ Industrial Protocols" icon="plug">
  Connect to virtually any industrial device or system with our extensive protocol support:
  
  - Industry-standard protocols: Modbus, OPC UA, OPC DA, BACnet
  - PLC-specific protocols: Siemens S7, Allen Bradley, Mitsubishi, Omron
  - IoT protocols: MQTT, AMQP, HTTP/REST
  - Database protocols: SQL, Oracle, PostgreSQL
</Card>

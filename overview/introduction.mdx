## What is Proxus?

Proxus is a next-generation Industrial IoT (IIoT) platform that unifies data collection, analysis, storage, enrichment, monitoring, and integration capabilities into a single powerful system. Purpose-built for high performance and scalability, Proxus transforms how organizations manage and leverage their industrial data.

## Platform Capabilities

### Data Collection

* **50+ OT Adapters** for seamless device connectivity

* **Flexible Protocol Support** including Modbus, OPC-UA, BACnet

* **Automatic Device Discovery** and configuration

* **Built-in Data Buffering** for reliability

### Real-time Processing

* **Rule Engine** for instant data analysis

* **Anomaly Detection** using advanced algorithms

* **Custom Functions** with C# support

* **Event-driven Architecture** for responsive actions

### Edge Computing

<Note>
  Deploy Proxus capabilities to edge devices for local processing and reduced latency
</Note>

* Full container support with Docker

* Local data processing and buffering

* Seamless synchronization with central platform

* Offline operation capability

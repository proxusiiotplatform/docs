# Welcome to Proxus

Proxus is a high-performance, distributed next-generation Industrial IoT Platform that unifies all essential features for industrial data management in a single system. It provides comprehensive capabilities for data collection, analysis, storage, enrichment, monitoring, and integration.

## What is Proxus?

Proxus serves as an end-to-end solution for managing industrial data workflows, offering:

- **Seamless Data Collection**: With support for over 50 OT protocols, making industrial data collection effortless
- **Real-time Analytics**: Process data streams with user-defined rules and trigger notifications, including anomaly detection
- **Enterprise-grade Storage**: Powered by TimescaleDB for reliable and scalable data management
- **Integrated Development**: Built-in IDE for C#-based development with deploy/undeploy capabilities
- **Visual Analytics**: Create powerful dashboards using the integrated designer with SQL-based data queries
- **Extensive Integration**: Connect with 20+ IT systems for seamless data flow
- **Edge Computing**: Deploy all features (except visualization) to edge devices
- **Container-Native**: Full Docker support, including edge computing services
- **Cloud-Ready**: Support for cloud-native deployments including Kubernetes
